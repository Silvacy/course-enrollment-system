# Course Enrollment System

A Java-based Course Enrollment and Grade Management System developed as part of an Object-Oriented Programming assignment. The project demonstrates the use of classes, objects, encapsulation, constructors, instance methods, static methods, collections, and exception handling.

## Features

- Add new courses
- Register students
- Enroll students in courses
- Assign grades to enrolled students
- Calculate a student's overall grade
- Prevent duplicate students and courses
- Prevent enrollment in full courses
- Handle invalid user input using exception handling

## Technologies

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- HashMap
- Scanner
- Exception Handling

## Project Structure

- **University** – Main class with the command-line interface.
- **Student** – Stores student information, enrolled courses, and grades.
- **Course** – Stores course information and manages enrollment capacity.
- **CourseManagement** – Provides static methods to manage students, courses, enrollments, and grades.

## How to Run

1. Download or clone this repository.
2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
3. Compile and run `University.java`.
4. Use the menu to manage courses, students, enrollments, and grades.
